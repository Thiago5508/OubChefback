# OubChefback
 Repositorio em Node/Typescript utilizando mongoDB não relacional.
 
 ## Tecnologias
- Node.js
- TypeScript
- Express
- MongoDB
- MonGoose ODM
- Criptoria e Segurança Hash

  ## Como rodar
- npm install
- npm run dev

## variaveis de Ambiente
variaveis foram removidas por questão de segurança!

## Feature: Access Control

Implementando controle de acesso utilizando como base Role-Based Access Control .

## Feature: Product creat/list

Implementação do sistema de criação de Produtos e listagem .

## Feature: Category creat/list

Implementação do sistema de criação das categorias, listagem e detalhamento .
